Marriott Search Project

Marriott Search was the second large project I created. It uses mainly Python and CSS.

Main features:

  • Form input is sent to Marriott.com with a separate thread.
  • The results are scraped using the Selenium framework and sent to a provided email.
  • Daily results can be requested by inputting a number in the Email Frequency box. The job is scheduled with the APScheduler Python library with Django.
  • Search results are stored in a Postgres database. The previous searches can be retrieved and deleted.

#30days30projects

#30days30projects is a challenge to complete 30 projects in 30 days. I participated because I thought it was a fun approach to improve my front-end skills including HTML, CSS, and Javascript. The process was fun. Ultimately, I was successful in completing 30 projects in 30 days.

#30days30projects Rules

  1. Create a project per day.
  2. A project could be an app, a component, a website, a game, a library, etc.
  3. Your choice of langauge and framework.
  4. The project is considered complete for the day by 11:59PM.
  5. Missed projects must be made up within 7 days.
